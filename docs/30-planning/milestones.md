# 里程碑

## M1 — 最小闭环

内容：插件壳 + AO3 解析 + 规则引擎 + 渲染 + 本地存储

验收：打开 AO3 列表页，写死的测试规则能生效（高亮 / 弱化 / 警示 / 折叠）。

## M2 — 页面交互

内容：hover 按钮 + 快速添加菜单 + Toast + 即时重渲染

验收：用户从 tag 上一键创建规则，页面立即反映效果。

## M3 — 管理界面

内容：popup 当前页统计 + options 规则 CRUD

验收：用户能完整管理已有规则（查看 / 新增 / 编辑 / 删除 / 启停）。

## M4 — 稳定性

内容：MutationObserver + 节流 + 错误兜底 + 占位条完善

验收：可以给真实用户安装使用，无明显卡顿或报错。

## MVP 完成 = M1 + M2 + M3 + M4
