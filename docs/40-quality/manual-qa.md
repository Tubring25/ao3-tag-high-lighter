# 手动测试清单

MVP 阶段 popup / options / 页面交互以手动测试为主。

## 测试环境

- Chrome 开发者模式加载扩展
- 打开 AO3 真实页面（搜索结果页、tag listing 页、作品详情页）

## 基本流程

- [ ] 安装扩展后 AO3 页面无报错
- [ ] 列表页 tag 能被正确解析（控制台无解析异常）
- [ ] 预设规则的高亮 / 弱化 / 警示 / 折叠效果正确
- [ ] hover 任意 tag 出现小按钮
- [ ] 点击按钮 → 弹出菜单 → 选择 action → 规则创建成功
- [ ] 规则创建后页面即时更新
- [ ] Toast 正常出现并消失
- [ ] Popup 显示当前页命中数量
- [ ] Popup 全局开关可切换，关闭后页面样式恢复
- [ ] Options 页面能查看 / 新增 / 编辑 / 删除 / 启停规则
- [ ] 刷新页面后规则仍然生效（持久化验证）

## 边界情况

- [ ] 无 tag 的作品不报错
- [ ] 同一 tag 命中多条规则时优先级正确
- [ ] 大量规则（50+ 条）时页面不卡顿
- [ ] AO3 页面动态加载内容后新内容也被处理
