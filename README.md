# AO3 Tag Highlighter

桌面端 Chrome 扩展，让 AO3 用户根据个人 tag 规则对作品进行高亮、警示、弱化或折叠。

## 状态

MVP 规划中，尚未开始编码。

## 功能

- 识别 AO3 页面中的 relationship / character / freeform tag
- 支持精确、包含、通配符三种匹配方式
- 四种规则效果：高亮 / 警示 / 弱化 / 折叠作品
- 页面内悬停快速添加规则
- Popup 查看当前页命中 + 全局开关
- Options 页面完整规则管理
- 本地存储，无需登录
